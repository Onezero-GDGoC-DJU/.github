# 일영 Github Organization

개발동아리 **일영**의 Github Organization입니다.  
프로젝트 협업, 코드 공유, 스터디 자료 정리 및 운영 문서 관리를 위해 사용합니다.

---

## 주요 목적
- 팀 프로젝트 협업
- 코드 리뷰 및 PR 관리
- 스터디 자료 아카이빙
- 운영 문서 정리

## 참여 방법
1. 운영진에게 Github username 전달
2. Organization 초대 수락
3. 팀별 repository 참여

## 협업 규칙
- main 브랜치 직접 push 금지
- 작업은 feature 브랜치에서 진행
- Pull Request 후 merge
- 리뷰 의견 반영 후 병합

## 링크
- 동아리 소개: 여기에 링크
- 디스코드: 여기에 링크
- 노션: 여기에 링크
