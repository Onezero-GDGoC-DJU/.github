## ✨ About Us

**일영**은 함께 배우고, 만들고, 성장하는 개발동아리입니다.  
단순히 코드를 작성하는 데서 끝나는 것이 아니라,  
팀 프로젝트와 스터디, 기록과 공유를 통해 **지속적으로 성장하는 협업 문화**를 지향합니다.

이 Github Organization은 그 과정에서 만들어지는  
여러 프로젝트와 문서, 스터디 자료와 협업 기록이 모이는 공간입니다.

---

## 💡 What We Do

저희는 주로 아래와 같은 활동을 진행합니다.

- 👨‍💻 팀 프로젝트 협업
- 📚 스터디 및 기술 공유
- 🔍 코드 리뷰와 피드백
- 🗂️ 프로젝트/문서 아카이빙
- 🤝 함께 배우는 협업 문화 만들기

---

## 🛠 Why Github Organization?

여러 명이 함께 하나의 프로젝트를 진행할 때는  
작업 흐름을 체계적으로 관리할 수 있는 협업 공간이 필요합니다.

Github Organization을 사용하면 다음과 같은 장점이 있습니다.

- 📦 프로젝트별 저장소를 체계적으로 관리할 수 있습니다.
- 👥 팀 단위 권한 설정이 가능합니다.
- 🔀 Pull Request를 통해 협업 과정을 기록할 수 있습니다.
- 📝 코드 리뷰와 피드백이 자연스럽게 이루어집니다.
- 📌 활동 기록이 남아 프로젝트 흐름을 추적하기 좋습니다.

---

## 🚀 How to Join

Github Organization 참여는 아래 순서로 진행됩니다.

1. Github 계정을 준비합니다.
2. 운영진에게 본인 **Github username** 을 전달합니다.
3. 초대 메일 또는 Github 알림을 확인합니다.
4. 초대를 수락하면 참여가 완료됩니다.

> 📍 이메일 주소가 아니라 **Github 아이디(username)** 를 보내야 합니다.

## 🤝 Our Collaboration Style

저희는 결과물만 올리는 공간보다,  
**과정을 함께 공유하는 협업 공간**을 더 중요하게 생각합니다.

- 작업 내용을 기록합니다.
- 리뷰를 통해 더 나은 방향을 찾습니다.
- 질문과 피드백을 자연스럽게 주고받습니다.
- 서로의 작업을 존중하며 함께 성장합니다.
